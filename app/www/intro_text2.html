QUICKSTART

 Visualization Only:

  1. For Molecular Level 1 (ML1) nodes: Input a text file with "score" and "id" variables.
  2. For Molecular Level 2 (ML2) nodes: Input a text file with "score" and "id" variable.
  3. For Mapping from ML1 to ML2: Input a text file with "from" and "to" variables.
  4. For Mapping within ML1: Choose ONE of the following:
    • For trivial edges, check "No Connections".
    • For sparse edges, input a text file with "from" and "to" variables.
    • For complete edges, check "Full Connections.
  5. For Mapping within ML2: Choose ONE of the following:
    • For trivial edges, check "No Connections".
    • For sparse edges, input a text file with "from" and "to" variables.
    • For complete edges, check "Full Connections.
  6. Choose layout and set thresholds for ML1 and ML2 nodes.
  7. Click generate graph to show visualization.

 Perturbation:

  1. Input a text file for population omics data for Ml1.
  2. Input a text file for associated population level phenotypic data.
  3. Input a text file mask of mapping from ML1 to ML2.
  4. Click "Run Model" to generate network.
  5. Click "Edit" to perturb nodes and edges.
